2. Local server port conflict
   - Error: "[Errno 10048] error while attempting to bind on address ('127.0.0.1', 8000): only one usage of each socket address..."

   - Fix: Started the app on port `8001` for local verification.
   - Result: The app is reachable at `http://127.0.0.1:8001/`.

3. Template engine import cleanup
   - Observation: `Jinja2Templates` and `Response` were imported but no longer used after the fix.
